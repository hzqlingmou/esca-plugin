# 安装教程
在Yunzai根目录执行即可
> [!tip]## 如果你的网络环境较差，可使用ghproxy安装
> ``` bash
  git clone --depth=1 https://mirror.ghproxy.com/https://github.com/hzqlingmou/esca-plugin ./plugins/esca-plugin
> ```
## GitHub安装(国外推荐)(更新较快)
``` bash
git clone --depth=1 https://github.com/hzqlingmou/esca-plugin ./plugins/esca-plugin
```
## 安装依赖
``` bash
pnpm install --filter=esca-plugin
```
